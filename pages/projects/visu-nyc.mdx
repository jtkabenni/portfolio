Visu-NYC is a geospacial visualization application that allows users to see public data from NYC. This app embeds a map component called [kepler.gl](https://kepler.gl/) and has backend functionality to allow users to create accounts and save maps to their profiles. The data is sourced from [NYC Open Data](https://opendata.cityofnewyork.us/).

# Callout Component

A built-in component to show important information to the reader.

import { Callout } from "nextra/components";

## Example

<Callout>
  A **callout** is a short piece of text intended to attract attention.
</Callout>

<Callout type="info">
  A **callout** is a short piece of text intended to attract attention.
</Callout>
